# Project Memory

## Core
IT project budget tracking app. Light theme, teal primary (187 72% 40%), dark slate sidebar.
Inter font. Professional PMO/finance aesthetic.
Roles: Admin, Office Admin, PM, Executive Viewer, Reporter.
Reporter can only see/add Timesheets on allocated projects. Auto-matched by email.
Baseline reporting currency is always EUR. Currencies: EUR, GBP, RON.
